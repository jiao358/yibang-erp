## 🚀 快速开始

### 开发环境准备
1. **后端环境**：Java 17+, Maven 3.8+, MySQL 8.0, Redis 7.x
2. **前端环境**：Node.js 18+, npm 9+, Vue 3
3. **数据库**：执行 `database/init/ddl/03_orders_tables.sql`

### 启动项目
```bash
# 后端启动
cd backend
mvn spring-boot:run

# 前端启动
cd frontend
npm run dev
```

### 访问地址
- **前端**：http://localhost:7101
- **后端API**：http://localhost:8080/api
